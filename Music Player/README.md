# 🎵 Music Player

A modern music player built using HTML, CSS, and JavaScript.

## 🚀 Features

- Play / Pause
- Previous / Next Song
- Playlist
- Progress Bar
- Volume Control
- Keyboard Shortcuts
- Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Music Player
│
├── index.html
├── style.css
├── script.js
├── songs
└── images
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html`.

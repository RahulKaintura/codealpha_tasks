# 🧮 Calculator

A responsive calculator built using HTML, CSS, and JavaScript.

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Keyboard Support
- Calculation History
- Responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Calculator
│
├── index.html
├── style.css
└── script.js
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html`.

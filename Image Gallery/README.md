# 🖼️ Image Gallery

A responsive Image Gallery built using HTML, CSS, and JavaScript.

## 🚀 Features

- Responsive Design
- Image Categories
- Image Filtering
- Lightbox Preview
- Previous / Next Navigation
- Keyboard Controls
- Hover Animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Image Gallery
│
├── index.html
├── style.css
├── script.js
└── images
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
